# BMKG Weather App (ADM4)

Aplikasi cuaca berbasis web menggunakan **API resmi BMKG**  
Lokasi otomatis hingga **desa/kecamatan (ADM4)**.

## Fitur
- Auto detect lokasi browser
- Prakiraan cuaca resmi BMKG
- Data per jam
- Tanpa API key berbayar

## Tech Stack
- Next.js (App Router)
- TypeScript
- BMKG Public API
- OpenStreetMap Nominatim
- Upstash Redis

## Setup
```bash
npm install
npm run dev
